# codingcamp-submission

Repository ini berisi seluruh hasil submisi saya selama mengikuti **Coding Camp by DBS x Dicoding**.

Setiap folder atau file dalam repository ini merupakan bagian dari tugas, tantangan, atau proyek yang diberikan selama program Coding Camp berlangsung.

## 📌 Tentang Coding Camp by DBS x Dicoding
Coding Camp ini merupakan hasil kolaborasi antara **Bank DBS Indonesia** dan **Dicoding Indonesia**, yang dirancang untuk meningkatkan kemampuan peserta dalam bidang teknologi, terutama di bidang pemrograman dan pengembangan perangkat lunak.
